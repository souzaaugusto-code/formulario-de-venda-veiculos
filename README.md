# 🚗 Formulário de Venda de Veículos

Um formulário simples para cadastro e venda de carros. Projeto feito para praticar a estrutura de tags no HTML, validação de campos obrigatórios e estilização com CSS.

## 🧠 O que foi praticado:
* **HTML:** Organização da estrutura de caixas (`divs`), inputs de texto, número, data, rádio, checkboxes e upload de arquivos.
* **Validação:** Uso de campos obrigatórios (`required`) para garantir que o formulário não seja enviado em branco.
* **CSS:** Ajustes de margens, alinhamento de colunas de opcionais lado a lado e personalização de cores.
